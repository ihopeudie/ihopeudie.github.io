Установка Java Компиляция Типы данных Коллекции

Установка Java

brew install java

Проверка версии Java

java --version

Установка SDKMAN

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"

Проверка версии SDKMAN

sdk version

Установка Java через SDKMAN

sdk install java x.y.z-zulu
