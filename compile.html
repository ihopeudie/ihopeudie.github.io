Компиляция класса

javac Main.java

Запуск скомпилированного класса

java Main

Запуск с аргументами

java Main arg0 arg1 arg2

Добавление библиотек в classpath

javac -cp myLib.jar:my2ndLib.jar Main.java

Создание jar файла

jar cf jar-file input-file(s)
jar cfm jar-file manifest-addition input-file(s)
java -jar JAR-file

Создание jar файла с манифестом и выбором главного класса

jar cfe app.jar MyApp MyApp.class
